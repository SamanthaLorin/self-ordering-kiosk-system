# Customer Module

This module handles the customer interface for the Self-Ordering Kiosk System. It allows customers to browse the menu, customize their orders, and send them directly to the admin system.

## Features
* **Interactive Menu:** Browse food categories with images and pricing.
* **Order Customization:** Add item modifiers (e.g., quantity).
* **Cart Management:** View, edit quantities, or remove items before checking out.

## Tech Stack & Requirements
* **Language:** Java
* **Build Tool:** Apache Ant (`build.xml`)
* **IDE Used:** [e.g., NetBeans]

## Directory Structure
* `src/` - Contains the Java source code files and GUI forms.
* `src/resources/` - Contains menu images, icons, and system assets.
* `src/images/` - Contains Food and Drink Icons
* `nbproject/` - Project configuration settings for NetBeans.
