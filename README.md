# Self-Ordering Kiosk System

A Java-based self-ordering kiosk system that allows customers to place orders while providing admin with tools to manage menu items and monitor transactions.

## Features

### Customer Module
- Browse menu
- Place orders
- View order summary
- Calculate total bill
- Add Discount

### Admin Module
- Add menu items
- Update menu/order items
- Remove menu/order items
- View customer orders
- View Most Ordered Items

## Used Tech

- Java
- Java Swing
- Object-Oriented Programming
