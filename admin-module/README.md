# Admin Module

This module serves as the backend management dashboard for the Self-Ordering Kiosk System. It allows the admin to manage the menu, track real-time orders coming from the customer kiosk, and view/edit orders and menu.

## Features
* **Dashboard/Sales Analytics:** View Most Recent Orders, Most Ordered Items, Total Orders Today, Amount of Pending/In Progress/Completed Orders.
* **Menu Management (CRUD):** Add, update, view, or delete menu items and prices. 
* **Stock Control:** Toggle item availability (e.g., mark an item as "Out of Stock" so that it won't be clickable in the customer kiosk).

## Tech Stack & Requirements
* **Language:** Java
* **Build Tool:** Apache Ant (`build.xml`)
* **IDE Used:** [e.g., NetBeans / VS Code]

## Directory Structure
* `src/` - Contains the Java backend source code and admin GUI panels.
* `nbproject/` - Project configuration settings for NetBeans.
